# Agentes del Proyecto

Este proyecto cuenta con 3 agentes especializados para asistir en diferentes áreas del desarrollo.

---

## Lista de Agentes

| Agente | Especialidad | Cuando consultarlo |
|--------|-------------|-------------------|
| [[Fisica-y-Mecanicas]] | Física y movimiento | Problemas con movimiento de nave/asteroides/proyectiles, colisiones, velocidad, ULTi |
| [[UI-y-Graficos]] | UI y gráficos | Problemas con pantallas (Game Over, Top 5), posicionamiento, efectos visuales, fuentes |
| [[Persistencia-y-Datos]] | Persistencia y datos | Problemas con Top 5, Firebase, datos corruptos, validación |

---

## Cómo Usar

Cuando tengas un problema o tarea específica:

1. **Identifica el área** del problema:
   - Física/Movimiento → Agente Fisica-y-Mecanicas
   - UI/Gráficos/Posicionamiento → Agente UI-y-Graficos
   - Datos/Firebase/Top 5 → Agente Persistencia-y-Datos

2. **Consulta el agente correspondiente** para obtener:
   - Conocimiento del código relevante
   - Ejemplos de implementación
   - Protocolo de trabajo

---

## Yo (Orquestador)

Yocoordinate los agentes ytaskso el flujo de trabajo general. Puedo:
- Identificar qué agente necesita cada tarea
- Consultar a un agente específico para tasks complejos
- Gestionar la comunicación entre agentes si es necesario
- Mantener la coherencia arquitectónica del proyecto
